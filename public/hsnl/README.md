# Hồ sơ năng lực (HSNL)

Đặt file PDF hồ sơ năng lực tại đây với tên chính xác:

    ho-so-nang-luc.pdf

Nút "Tải xuống HSNL (PDF)" ở trang `/services` (Giải pháp) đang trỏ tới:

    /hsnl/ho-so-nang-luc.pdf

Chỉ cần copy file PDF vào thư mục này và đặt đúng tên là nút tải sẽ hoạt động.
Nếu muốn đổi tên file, sửa đường dẫn trong: `src/app/services/ServicesClient.tsx`
